# MATLAB and Simulink

This folder contains the ECG input data, MATLAB preprocessing script, and signal/result images.

## Files

| File | Purpose |
| --- | --- |
| `ECG1.m` | Loads ECG data and creates clean/noisy ECG plots |
| `100m.mat` | ECG data file used by the MATLAB script |
| `clean.jpg` | Clean ECG output image |
| `ecg_noisy.jpg` | Noisy ECG output image |
| `simulink_run.jpg` | Simulink run/output screenshot |

## Run

Open MATLAB in this folder and run:

```matlab
ECG1
```

The script currently uses an SNR value of 10 dB.
